Portfolio

This portfolio showcases six of my python data projects that I have pinned for easy access. Each project represents a distinct data problem that I have solved through the use of various data science techniques such as data wrangling and machine learning. You can find more by clicking on the Github link to access my repository.

Mushroom Supervised Classification Project (part of IBM ML certification)

The Mushroom Supervised Classification Project was part of the IBM ML certification program. The aim was to use supervised classification models to determine if mushrooms were poisonous. The data was sourced from Kaggle and explored using EDA techniques. Sklearn's LabelEncoder was used to encode categorical data and a test-train split was performed. Logistic Regression, KNeighborsClassifier, and DecisionTreeClassifier models were used.

Bjj classification and Extraction

The project aims to automate the analysis of Brazilian Jiu-Jitsu data and make predictions using supervised classification models. Data was webscraped from a reputable source for Bjj, and several libraries such as pandas, numpy, and sklearn were used for data frame creation, manipulation, and machine learning models. The notebook works by taking user inputs to scrape fighter data, clean it, and analyze it using machine learning models, with potential for further optimization

IBM Course Recommendation system

The project aims to build a course recommender system using various machine learning models. Both supervised and unsupervised models were used and compared with each other. The machine learning models used include dot product, Bag of Words, clustering, PCA, KNN, NMF, and a Tensorflow neural network classifier with embeddings.

Pyspark Basics

This notebook covers PySpark Basics including its architecture, installation, setup and configuration It then discusses SQL Data Wrangling with PySpark, including reading and writing data from/to various sources, filtering, joining, grouping, and aggregating data Finally, the notebook discusses machine learning with PySpark, covering topics such as feature engineering, model training, tuning, evaluation, and deployment.

Reddit NLP analysis

This is a Python script that extracts data from the Reddit API, performs data cleaning and analysis using NLP techniques, and performs sentiment analysis on the text. The purpose of the project is to analyze the sentiment of posts related to Brazilian Jiu-Jitsu (BJJ) and identify the most frequently mentioned people and topics. The project also includes the development of a dashboard that links to the output csv and images.

Credit Card Fraud Detection

The project aimed to use Deep Learning (specifically autoencoding) to classify credit card transactions as fraudulent or not. The dataset used was considerably large with 284,807 rows and 31 columns, and was imbalanced with only 492 fraud transactions out of the total. Various normalization methods and hyperparameters for the autoencoder were tested in conjunction with three supervised classification models, resulting in the top three models having an accuracy of 0.99, 0.96, and 0.96.
